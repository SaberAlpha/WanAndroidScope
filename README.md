# WanAndroidScope
- 该项目基于 [鸿洋](https://github.com/hongyangAndroid) 大佬的 [玩Android](https://www.wanandroid.com/blog/show/2) 接口
- 项目采用 Kotlin 语言
- 整体采用  MVVM 架构编写
- 使用到的 JetPack组件有 LiveData 、 ViewModel 、 Lifecycle 、 DataBinding 、 Navigation
- 使用协程Flow代替rxjava,开发更加顺滑

### 已有功能

- 首页最新文章
- 公众号列表
